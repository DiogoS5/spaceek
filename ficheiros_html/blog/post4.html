Primeiros Passos Técnicos: Aplicação Web, Algoritmos e Caixa 3D

Beatriz Guia Maio 2025

Esta semana, a nossa equipa reuniu-se para organizar as tarefas, especialmente tendo em conta que ainda não recebemos todo o material — incluindo um sensor crucial, essencial para avançarmos com os testes.

Para aproveitar este período de espera, o nosso colega Gonzalo tomou a iniciativa de começar a desenhar uma caixa personalizada para impressão 3D. Esta estrutura irá alojar o sensor principal, o microcontrolador ESP32, bem como os sensores de temperatura e ruído. As imagens abaixo mostram o progresso feito até ao momento.

Entretanto, eu (Beatriz Guia) e o meu colega Tiago Gomes demos início ao desenvolvimento da aplicação web. Após discutirmos a estrutura da plataforma, decidimos construir a aplicação com Java para o backend e HTML e CSS para o frontend.

O nosso colega Daniel Burke está a investigar como podemos enviar dados através da rede eduroam, enquanto o Miguel Fialho e o Diogo Sousa estão atualmente concentrados no desenvolvimento do algoritmo de deteção de entradas e saídas nas salas de estudo.
