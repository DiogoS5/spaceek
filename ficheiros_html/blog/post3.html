Desenvolvimento da WebApp: A Criar uma Experiência Intuitiva para os Utilizadores

Beatriz Guia Maio 2025

Nos últimos dias, eu, Beatriz Guia, juntamente com o meu colega Tiago Gomes, estivemos dedicados ao desenvolvimento da aplicação web do projeto Spaceek. O nosso principal objetivo é garantir que todas as funcionalidades sejam apresentadas de forma clara, eficiente e intuitiva para os utilizadores.

Estamos a trabalhar com HTML, CSS e JavaScript, tanto no frontend, responsável pela interface que os estudantes irão utilizar, como no backend, que trata da lógica e da comunicação com os dados.

Como parte do progresso, já conectámos a aplicação à Firebase, o que nos permite receber e armazenar dados em tempo real. Para testar esta integração e garantir que tudo funciona corretamente, criámos dados fictícios que simulam os valores de ocupação, ruído e temperatura. Assim que o protótipo estiver operacional, faremos a transição para os dados reais recolhidos pelos sensores.

Este passo marca um avanço importante no projeto, aproximando-nos de uma versão funcional e pronta para testes em contexto real!
